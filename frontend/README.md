# Tartua Frontend

This folder contains a React + Vite starter for the Tartua creator dashboard.

## Setup

1. Change directory into `frontend`
2. Run `npm install`
3. Run `npm run dev`

The app starts on `http://localhost:4173` by default.

## Notes

- Authentication and API integration are not wired yet.
- The frontend is scaffolded with placeholder demo pages and layout.
- Update `frontend/src/api/api.ts` to connect to the backend.
